# New Preset Quality Checklist

For every new preset before merging:

- [ ] Defined in domain.md if a new shot type
- [ ] Parameter count is 5–7 (push back hard on more)
- [ ] Default values produce a usable shot with no tweaking
- [ ] Thumbnail generated and looks like the shot it represents
- [ ] Stretches to half duration without breaking
- [ ] Stretches to double duration without breaking
- [ ] Beat-sync (if supported) places key moments on beats
- [ ] Bakes to clean curves
- [ ] Inspector shows only relevant parameters
- [ ] Reviewed through cinematographer or motion designer lens (whichever fits)
